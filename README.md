#### Описание лабораторной  работы
Проект представляет собой реализацию различных алгоритмов сортировки, 
математических функций и структур данных для сравнения их производительности.

#### Основной функционал:
- **Алгоритмы сортировки**: пузырьковая, быстрая, count sort, radix sort, bucket sort
- **Математические функции**: вычисление чисел Фибоначчи и факториала (итеративно и рекурсивно)
- **Структуры данных**: реализация стека на основе списка
- Сравнение скорости работы различных алгоритмов сортировки

src/main.py: Главный модуль для сравнения производительности алгоритмов сортировки:
- Тестирует 6 алгоритмов сортировки на массивах размером 100, 500 и 1000 элементов
- Сравнивает со встроенной функцией sorted()
- Выводит время выполнения каждого алгоритма

src/sorting.py: Реализации алгоритмов сортировки:
- bubble_sort() - пузырьковая сортировка
- quick_sort() - быстрая сортировка
- counting_sort() - сортировка подсчетом
- radix_sort() - поразрядная сортировка
- bucket_sort() - блочная сортировка (для float)
- heap_sort() - сортировка кучами

src/fibonacci_factorial.py:
Математические функции:
- fibo() - числа Фибоначчи (итеративно)
- fibo_recursive() - числа Фибоначчи (рекурсивно)
- factorial() - факториал (итеративно)
- factorial_recursive() - факториал (рекурсивно)

src/stack.py:
Реализация структуры данных "Стек" и "Очередь":
- push() - добавление элемента
- pop() - удаление элемента
- peek() - просмотр верхнего элемента
- is_empty() - проверка на пустоту


- enqueue() - добавление элемента в конец очереди
- dequeue() - удаление и возврат элемента из начала очереди
- front() - просмотр первого элемента без удаления
- is_empty() - проверка на пустоту

Тестирование
- test_fibonacci_factorial.py - тесты математических функций
- test_sorting.py - тесты всех алгоритмов сортировки
- test_stack.py - тесты структуры данных Stack и Queue
